# FAQ - Часто задаваемые вопросы

## Авторизация и аккаунт

### Как войти в систему?

Вы можете войти несколькими способами:

1. **Email и пароль** - стандартная форма входа
2. **Google OAuth** - кнопка "Войти через Google"
3. **GitHub OAuth** - кнопка "Войти через GitHub"

### Не работает авторизация через Google

Если при входе через Google появляется ошибка:

1. Очистите кэш и cookies браузера
2. Проверьте, что у вас включены cookies для сторонних сайтов
3. Попробуйте другой браузер
4. Убедитесь, что ваш Google-аккаунт подтверждён

Если проблема сохраняется - напишите в поддержку, укажите код ошибки.

### Забыл пароль

1. На Strange входа нажмите "Забыли пароль?"
2. Введите your email
3. Проверьте почту (включая папку Спам)
4. Перейдите по ссылке в письме
5. Установите новый пароль

### Не приходит письмо с подтверждением

Проверьте:

- Папку "Спам" / "Промоакции"
- Правильность указанного email
- Не заблокирован ли домен @legion.app в настройках почты

Письмо должно прийти в течение 5 минут. Если не пришло - запросите повторную отправку или свяжитесь с поддержкой.

## Тарифы и оплата

### Какие тарифы доступны?

**Free** (Бесплатный):

- 100 запросов в месяц
- Базовые модели
- Email поддержка

**Pro** ($29/месяц):

- 10,000 запросов в месяц
- Все модели, включая GPT-4
- Приоритетная поддержка
- API доступ

**Enterprise** (от $299/месяц):

- Неограниченные запросы
- Выделенные ресурсы
- SLA 99.9%
- Персональный менеджер
- Кастомные интеграции

### Как перейти на платный тариф?

1. Откройте Settings → Billing
2. Выберите нужный тариф
3. Нажмите "Upgrade"
4. Заполните платёжные данные
5. Подтвердите оплату

Поддерживаемые способы оплаты: Visa, Mastercard, PayPal.

### Есть ли скидки?

Да! При годовой оплате действует скидка **20%**.

Для студентов и некоммерческих организаций - скидка **50%** (требуется подтверждение).

### Что будет при превышении лимита?

**Free план**: запросы временно блокируются до следующего месяца. Можно апгрейдиться на Pro.

**Pro план**: дополнительные запросы тарифицируются по $0.01 за запрос.

**Enterprise**: лимита нет.

### Можно ли вернуть деньги?

Да, действует политика возврата в течение **14 дней** без объяснения причин.

## Функции и возможности

### Как экспортировать данные?

1. Откройте раздел с нужными данными
2. Нажмите Settings (⚙️) → Export
3. Выберите формат: CSV, JSON, или Excel
4. Нажмите "Download"

Экспорт доступен в тарифах Pro и Enterprise.

### Как добавить члена команды?

1. Settings → Team
2. Нажмите "Invite Member"
3. Введите email нового участника
4. Выберите роль (Admin, Member, Viewer)
5. Отправьте приглашение

Новому участнику придёт письмо с приглашением.

### Какие роли доступны в команде?

- **Admin**: полный доступ, управление биллингом
- **Member**: создание и редактирование проектов
- **Viewer**: только просмотр

### Как настроить API доступ?

1. Settings → API Keys
2. Нажмите "Generate New Key"
3. Задайте имя и scope (права доступа)
4. Скопируйте ключ (показывается только один раз!)
5. Используйте в заголовке: `Authorization: Bearer YOUR_KEY`

Документация API: https://docs.legion.app/api

### Поддерживается ли интеграция с GitHub?

Да! Вы можете:

- Импортировать репозитории в базу знаний
- Использовать код из GitHub для контекста
- Автоматически обновлять knowledge base при push

Настройка: Settings → Integrations → GitHub

## Производительность

### API отвечает медленно

Возможные причины:

1. **Высокая нагрузка** - пиковое время (11:00-15:00 MSK)
2. **Большой запрос** - уменьшите размер контекста
3. **Медленная модель** - попробуйте более быструю (GPT-3.5 вместо GPT-4)

Если timeout > 10 секунд на Enterprise плане - свяжитесь с техподдержкой немедленно.

### Как ускорить ответы?

- Используйте stream режим для progressive loading
- Оптимизируйте промпты (удалите лишнее из контекста)
- Используйте кэширование для повторяющихся запросов
- Включите CDN для статических ресурсов (Enterprise)

## Безопасность и приватность

### Где хранятся мои данные?

Все данные хранятся в зашифрованном виде на серверах в EU (Германия, Франкфурт).

### Используется ли мой контент для обучения моделей?

**Нет!** Ваши данные не используются для обучения AI моделей. Это гарантировано нашей политикой конфиденциальности.

### Можно ли удалить все свои данные?

Да. Settings → Account → Delete Account. Все данные будут безвозвратно удалены в течение 48 часов.

## Часто встречающиеся ошибки

### Error 500: Internal Server Error

Временная проблема на сервере. Подождите 5 минут и попробуйте снова. Если не помогло - свяжитесь с поддержкой.

### Error 401: Unauthorized

Ваша сессия истекла. Выйдите и войдите заново.

### Error 429: Too Many Requests

Вы превысили лимит запросов. Подож дайте немного или апгрейдьтесь на более высокий тариф.

### Error 403: Forbidden

У вас нет доступа к этому ресурсу. Проверьте права доступа или свяжитесь с администратором команды.
